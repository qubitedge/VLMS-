Experiment: Hello World
--------------------------------------------------------
[Pre-test Questions]
1. Which of the following is the correct extension for a C source file?
   a) .cp
   b) .c
   c) .cpp
   d) .cv
   Answer: .c

2. Which header file is required to use printf() in C?
   a) math.h
   b) string.h
   c) stdio.h
   d) stdlib.h
   Answer: stdio.h

3. What is the entry point of every C program?
   a) start()
   b) begin()
   c) main()
   d) run()
   Answer: main()

4. What does \n do inside printf()?
   a) Prints the letter n
   b) Adds a tab space
   c) Moves to the next line
   d) Ends the program
   Answer: Moves to the next line

5. What value should main() return to indicate successful execution?
   a) 1
   b) -1
   c) 0
   d) NULL
   Answer: 0

[Post-test Questions]
1. What is the output of printf("Hello\nWorld");?
   a) Hello World
   b) Hello\nWorld (on two lines)
   c) HelloWorld
   d) Error
   Answer: Hello\nWorld (on two lines)

2. Which of the following correctly prints "JNTUGV" in C?
   a) print("JNTUGV")
   b) printf[JNTUGV]
   c) printf("JNTUGV");
   d) Printf("JNTUGV");
   Answer: printf("JNTUGV");

3. What happens if you remove #include<stdio.h> from the Hello World program?
   a) Program still runs normally
   b) printf() will not be recognized, compiler error
   c) Program prints nothing
   d) Program crashes at runtime
   Answer: printf() will not be recognized, compiler error

4. What does int before main() indicate?
   a) main() takes integer arguments
   b) main() returns an integer value
   c) main() is an integer variable
   d) Nothing, it is optional
   Answer: main() returns an integer value

5. Which of the following will print Hello World with a newline at the end?
   a) printf("Hello World")
   b) printf("Hello World\n")
   c) printf(\nHello World)
   d) printf("Hello World\t")
   Answer: printf("Hello World\n")
